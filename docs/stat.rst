=================
stat
=================

The ``stat`` command produces a table displaying directory filetype proportions


Usage
=============================================

.. code-block:: bash

    cparse stat [-a] <path>

Positional Arguments
=============================================
+------------+----------------+
| ``<path>`` | root directory |
+------------+----------------+


Optional Arguments
=============================================
+------------------+-----------------------------------------------------+
| ``-a``           | include hidden files                                |
+------------------+-----------------------------------------------------+
