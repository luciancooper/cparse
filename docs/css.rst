=================
css
=================

The ``css`` command parses css code

Usage
-----------------------------------------

``cparse css [-h] [-g] [-c] path``

Positional Arguments
"""""""""""""""""""""""""

+----------+---------------------+
| ``path`` | a css file to parse |
+----------+---------------------+

Optional Arguments
"""""""""""""""""""""""""

+--------+----------------------------------------------+
| ``-g`` | group identical selector property blocks     |
+--------+----------------------------------------------+
| ``-c`` | condense redundancies within property blocks |
+--------+----------------------------------------------+
